
# How to include GameLib in your project

## `plugin.yml`

```yaml
depend: [GameLib]
depends: [GameLib]
```

## `pom.xml`

```xml
<dependency>
    <groupId>com.pythoncraft.gamelib</groupId>
    <artifactId>gamelib</artifactId>
    <version>1.0</version>
    <scope>provided</scope>
</dependency>
```
